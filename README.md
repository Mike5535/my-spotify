## Основные комманды

### `npm start`

Запускает приложение без сжатия.
Откройте [http://localhost:3001](http://localhost:3001) чтобы увидеть его в браузере.

### `npm run build`

Собирает приложение сжимая файлы.

### `npm run test`

Запускает unit тесты на jest.
